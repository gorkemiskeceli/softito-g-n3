📘 JavaScript Sözlük

1. map()

const sayilar = [1, 2, 3, 4, 5];
const ikiKati = sayilar.map(sayi => sayi * 2);
console.log(ikiKati);
Çıktı: [2, 4, 6, 8, 10]
→ Dizideki her elemanı işleyip yeni bir dizi döndürür.

2. find()

const sayilar2 = [10, 20, 30, 40, 50];
const bulunan = sayilar2.find(sayi => sayi > 25);
console.log(bulunan);
Çıktı: 30
→ Koşulu sağlayan İLK elemanı döndürür.

3. reduce()

const sayilar3 = [1, 2, 3, 4, 5];
const toplam = sayilar3.reduce((acc, sayi) => acc + sayi, 0);
console.log(toplam);
Çıktı: 15
→ Diziyi tek bir değere indirger (toplama, çarpma vb.).

4. filter()

const sayilar4 = [5, 12, 8, 130, 44];
const buyukler = sayilar4.filter(sayi => sayi > 10);
console.log(buyukler);
Çıktı: [12, 130, 44]
→ Koşulu sağlayan tüm elemanlarla yeni dizi oluşturur.

5. some()

const sayilar5 = [1, 2, 3, 4, 5];
const varMi = sayilar5.some(sayi => sayi > 4);
console.log(varMi);
Çıktı: true
→ Koşulu sağlayan en az bir eleman var mı? (true/false)

6. every()

const sayilar6 = [2, 4, 6, 8, 10];
const hepsiCiftMi = sayilar6.every(sayi => sayi % 2 === 0);
console.log(hepsiCiftMi);
Çıktı: true
→ Tüm elemanlar koşulu sağlıyor mu? (true/false)

7. push()

const meyveler = ["elma", "armut"];
meyveler.push("çilek");
console.log(meyveler);
Çıktı: ["elma", "armut", "çilek"]
→ Dizinin sonuna yeni eleman ekler.

8. includes()

const renkler = ["kırmızı", "yeşil", "mavi"];
console.log(renkler.includes("yeşil"));
Çıktı: true
→ Dizi/string belirtilen elemanı içeriyor mu? (true/false)

9. toLowerCase()

const mesaj = "MERHABA DÜNYA";
console.log(mesaj.toLowerCase());
Çıktı: "merhaba dünya"
→ String'deki tüm harfleri küçük yapar.

10. toUpperCase()

const mesaj2 = "merhaba dünya";
console.log(mesaj2.toUpperCase());
Çıktı: "MERHABA DÜNYA"
→ String'deki tüm harfleri büyük yapar.

11. trim()

const bosluklu = " JavaScript öğreniyorum ";
console.log(bosluklu.trim());
Çıktı: "JavaScript öğreniyorum"
→ String'in başındaki ve sonundaki boşlukları siler.

12. split()

const cumle = "elma,armut,muz";
console.log(cumle.split(","));
Çıktı: ["elma", "armut", "muz"]
→ String'i belirtilen ayraca göre diziye böler.

13. join()

const kelimeler = ["JavaScript", "çok", "eğlenceli"];
console.log(kelimeler.join(" "));
Çıktı: "JavaScript çok eğlenceli"
→ Diziyi belirtilen ayraçla birleştirip string yapar.

14. parseFloat()

const fiyat = "49.99";
console.log(parseFloat(fiyat));
Çıktı: 49.99
→ String'i ondalıklı sayıya çevirir.

15. setInterval() & clearInterval()

let sayac = 0;
const interval = setInterval(() => {
  sayac++;
  console.log(sayac);
  if (sayac === 3) clearInterval(interval);
}, 1000);
Çıktı: 1, 2, 3 (1 sn aralıklarla)
→ setInterval() belirtilen sürede bir tekrarlayan işlem başlatır, clearInterval() durdurur.
